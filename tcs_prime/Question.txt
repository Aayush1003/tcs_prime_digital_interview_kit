** TIPS FOR TCS PRIME INTERVIEW **

1. TIME MANAGEMENT:
   - Solve the coding problem completely within 30-40 minutes
   - Test your solution with edge cases
   - Explain your approach clearly

2. COMMUNICATION:
   - Explain your thought process while coding
   - Ask clarification questions if problem is unclear
   - Discuss trade-offs in your solution

3. CODE QUALITY:
   - Write clean, readable code
   - Use meaningful variable names
   - Add comments where necessary
   - Handle edge cases

4. PROBLEM SOLVING:
   - Start with brute force approach
   - Then optimize to better time/space complexity
   - Discuss multiple approaches if possible

5. HR INTERVIEW:
   - Research about TCS before interview
   - Prepare 2-3 questions to ask the interviewer
   - Show enthusiasm and genuine interest
   - Be honest about your strengths and weaknesses

6. GENERAL TIPS:
   - Get good sleep night before interview
   - Arrive 10-15 minutes early
   - Make eye contact and smile
   - Be confident but humble
   - Follow up with thank you message
